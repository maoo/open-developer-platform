[CodeClimate](https://codeclimate.com/) delivers automated code review for test coverage, complexity, duplication, security, style and more; several [languages and platforms](https://docs.codeclimate.com/docs/programming-languages-frameworks) are supported.

Below are listed the main features that have been adopted by Foundation projects.

1. Integration with CI environments
2. Measures test coverage
3. Scans code for security vulnerabilities
4. Scans code for bad practices
5. Scans code for bugs

Simply define a [.codeclimate.yml](https://docs.codeclimate.com/v1.0/docs/example-codeclimateyml) in the root of your project to get started.
